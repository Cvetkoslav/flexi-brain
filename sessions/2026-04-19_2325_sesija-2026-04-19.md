### 4. Labeler implementacija (facebook.py + app.py + dragana.py)

Novi kod u `facebook.py`:
- `_get_all_labels()` — dohvata sve FB custom labele
- `_resolve_label_id(env_val, name_hint)` — env var ili fuzzy pretraga po imenu
- `apply_label(sender_id, label_id)` — POST na Graph API
- `remove_label(sender_id, label_id)` — DELETE na Graph API
- `sync_label(sender_id, stanje)` — mapiranje stanje → labela

Mapiranje:
- `lead` ← POCETAK / PRIKUPLJANJE / PREPORUKA / UBJEDJIVANJE
- `ordered` ← ZATVARANJE / PORUCIO / ZATVORENO

Trigger:
- Posle svakog Draginog odgovora (u `agents/dragana.py`)
- 2x dnevno za sve klijente (APScheduler, 8h i 20h u `app.py`)

Config: `LABEL_LEAD_ID` i `LABEL_ORDERED_ID` env var (opcioni — bot ih pronalazi automatski po imenu "kupovina" / "poručio").
